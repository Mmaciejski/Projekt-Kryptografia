# Kryptografia

Projekt dotyczący praktycznego wykorzystania kryptografii.

## Cel projektu
Celem projektu jest poznanie mechanizmów szyfrowania danych oraz podpisów cyfrowych
z wykorzystaniem narzędzi GPG i OpenSSL.

## Zakres
- generowanie kluczy kryptograficznych
- szyfrowanie i deszyfrowanie danych
- podpis cyfrowy i weryfikacja podpisu
- analiza bezpieczeństwa

## Narzędzia
- GPG
- OpenSSL
